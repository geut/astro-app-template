# Astro App Template

A starter Astro template for app development. Use this as a foundation for your next projects.

This is a basic app template with a sidebar for navigation, fully responsive for mobile devices.

## What's Included

- **[Astro](https://astro.build/)**: Static site generation with component islands
- **[React](https://reactjs.org/)**: Interactive components
- **[shadcn/ui](https://ui.shadcn.com/)**: Accessible UI components
- **[Tailwind CSS v4](https://tailwindcss.com/)**: Utility-first styling
- **[TypeScript](https://www.typescriptlang.org/)**: Type-safe development
- **[Lucide Icons](https://lucide.dev/)**: Consistent iconography
- **[oxlint](https://oxc.rs/docs/learn/tools/linter)**: Fast linter for JavaScript/TypeScript
- **[oxfmt](https://oxc.rs/docs/learn/tools/formatter)**: Code formatter

## Getting Started

```bash
# Clone the template
git clone https://github.com/geut/astro-app-template.git my-project
cd my-project

# Install dependencies
bun install

# Start development server
bun run dev
```

Open [http://localhost:4321](http://localhost:4321) to view your app.
